Vehicle Rental Management System API

About the API

Welcome to the Vehicle Rental Management System API. This API allows you to interact with our vehicle rental platform, including user registration, login, browsing available vehicles, and booking rentals.

API Endpoints

  • POST /register: Register a new user
  • POST /login: Login as an existing user
  • GET /vehicles: Retrieve a list of available vehicles
  • POST /bookings: Create a new booking
  • GET /bookings: Retrieve user bookings
  • POST /payments: Process a payment
